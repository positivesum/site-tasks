Allows to create and manage page specific tasks.

== Description ==

Site Tasks allows site contributors to create post specific tasks.
These tasks can be assigned to other users. This plugin features 
Pivotal Tracker integration which makes it easy to import tasks from
this plugin into Pivotal Tracker.
